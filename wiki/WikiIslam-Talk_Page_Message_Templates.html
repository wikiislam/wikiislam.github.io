Talk page message templates can be used for repetitive user notices. If editors choose to use them, they can also ensure a clear, consistent, polite and to-the-point tone.

List of Templates

The "Start", "Middle" and "End" sections contain templates that when placed together (usually one from each section) will form a complete talk page comment. They can be mixed and matched, and can also be incorporated into larger, non-template based messages (if this is done, the "End" template must be used to sign in order to let others know).

Areas marked by ### indicate template parameters where text is needed to be inserted manually, usually an article or page name that can be copy/pasted. There is no need to add square brackets for creating links as this is done automatically. Nor is there any need to sign messages (this is also done automatically by the "End" template).

Start

Existing User

For greeting existing users on their talk page. Name of user is automatically inserted in all "Start" templates when used on the target user's talk page:

Template:Utalk-greet: {{subst:Utalk-greet}}

Hi Ahmad.

New User

For greeting new users for the first time on their talk page. This templated does not require the use of a "Middle" template and can serve as a general welcome:

Template:Utalk-greet-new: {{subst:Utalk-greet-new}}

Hi Ahmad and welcome to WikiIslam, the online resource on Islam. Site policies and guidelines can be viewed here. If you need further clarification on an issue, let me or one of the administrators know. 

Middle

Talk Page

Signature

Template:Utalk-sign‎: {{subst:Utalk-sign‎}}

Unfortunately, your talk page comment(s) were not signed correctly. Please read WikiIslam:Signatures for help with how to sign comments. 
Indentation

Template:Utalk-indent: {{subst:Utalk-indent}}

Unfortunately, your talk page comment(s) were not indented correctly. Please read WikiIslam:Indentation for help with how to indent comments. 
Continuity

Template:Utalk-continuity: {{subst:Utalk-continuity|###}}

For continuity of discussion, comments should be kept on the same talk page where they were initiated. For the same reason, the creation of multiple discussions concerning the same topic should be avoided. Please continue this discussion at ###. 

Notices

Moved Content

Template:Utalk-content-moved: {{subst:Utalk-content-moved|###|###}}

A page or section that you created or were involved in editing has been moved from ### to ###. Please note the new location. 
Discussion Invite

Template:Utalk-join-discussion: {{subst:Utalk-join-discussion|###|###}}

There is currently a discussion titled "###" taking place at ### that you may be interested in. Please join the discussion and add your thoughts. 

Content

Reliable Sources

Template:Utalk-reliable-sources: {{subst:Utalk-reliable-sources|###}}

Unfortunately, your additions to ### were not based on reliable, published sources. Please read WikiIslam:Reliable Sources for help with identifying reliable sources. 
Citing Sources

Template:Utalk-cite-sources: {{subst:Utalk-cite-sources|###}}

Unfortunately, your additions to ### were not cited correctly. Please read WikiIslam:Citing Sources for help with how to cite sources. 
Formatting

Template:Utalk-format: {{subst:Utalk-format|###}}

Unfortunately, your additions to ### were not formatted correctly. Please read WikiIslam:Formatting for help with how to format wiki pages. 
Style, Tone or Content

Template:Utalk-style-tone-content: {{subst:Utalk-style-tone-content|###}}

Unfortunately, your additions to ### did not follow the site's rules concerning style, tone or content. Please read the Article Style and Content Guide for help with understanding what is or is not appropriate within articles. 
Standardization

Template:Utalk-standardization: {{subst:Utalk-standardization|###}}

Unfortunately, your additions to ### did not follow the site's rules concerning standardization. Please read WikiIslam:Standardization for help with how to standardize content. 
Repeated Issues

These should generally be used when an editor has not acknowledged the first and/or second message. They must be used in conjunction with one of the other "Content" templates listed above, but do not have to use a "Start" template (since they are most likely to be used in an ongoing discussion).

Second Notice

Template:Utalk-second-notice: {{subst:Utalk-second-notice}}

Is there something in particular that you're having difficulty with or would like explained? If that's the case, let me or one of the administrators know and we'll do our best to help. However, please try to avoid these problems in future edits or they may be reverted rather than corrected. 
Third Notice

Template:Utalk-third-notice: {{subst:Utalk-third-notice}}

Your edit was reverted because an editor has had to point out the same error multiple times to you. We appreciate your contributions to the site, but you cannot continue to make edits with the expectation that someone else will fix the problems those edits may have caused. Please engage in discussion before continuing your edits. Let us know how we can help you. 

End

For ending a message. There is no need to sign by typing 4 tildas (this is done automatically). This template must always be used to end a message if that message uses any of the other templates on this page:

Template:Utalk-end: {{subst:Utalk-end}}

Thanks. -- Amina (talk) 23:17, 13 March 2014 (PDT) (this message includes the use of talk page templates)

Other Templates

Unblock

After unblocking for an incorrect block by Edit Monitor, use {{subst:MediaWiki:Edit Monitor Admin/Unblock message}} on the user's talk page.
